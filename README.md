# Book scraper

This is a simple book scraper that scrapes the [To Scrape](http://books.toscrape.com/) website and stores the results in a CSV file.

## Setup

1. Clone the repository
2. Create a virtual environment
3.Install the requirements:
 run 'pip install -r requirements.txt'in the project directory in the terminal
4. Run the tests
5. Run the main script (python main.py)

## Running the tests

Run the tests with `pytest`.

```bash
$ pytest
```

## Running the scraper

Run the scraper with `python`.

```bash
$ python scraper.py
```
